# Music Folder

Ovaj folder sadrži pozadinsku muziku za web stranicu.

## Datoteke:
- `background-music.mp3` - Tiha, umirujuća pozadinska muzika

## Korištenje:
1. Preuzmite željenu muziku kao MP3 format
2. Preimenujte je u `background-music.mp3`
3. Postavite je u ovaj folder
4. Muzika će se automatski puštati na svim stranicama

## Napomenka:
- Glasnoća je postavljena na 30% (tiho)
- Muzika se ponavlja beskonačno
- Zahtijeva korisničku interakciju zbog browser autoplay politike
